# ManecHic.us – static copy ready for Cursor

Struktura:
- `index.html` – główny plik strony do edycji w Cursorze.
- `assets/css/` – dodatkowe style (np. `fonts.css` z Google Fonts).
- `assets/js/` – (obecnie puste) miejsce na własne skrypty.
- `assets/img/` – wszystkie lokalne obrazy (pobrane ze Static Showit / innych CDN).
- `assets/fonts/` – lokalne fonty webowe.
- `assets/media/`, `assets/files/` – inne pliki pomocnicze.
- `vendor/` – zewnętrzne biblioteki JS/CSS (jQuery, showit.js, itp.).

Ścieżki w `index.html` zostały zaktualizowane, aby wskazywały na lokalne zasoby.
Projekt możesz wrzucić do Cursora jako zwykły folder lub zip i dalej edytować.
